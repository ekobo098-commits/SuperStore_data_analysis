# 📊 Superstore Sales Data Analysis using SQL & Power BI

## Overview

This project analyzes the Superstore dataset using **PostgreSQL** for data analysis and **Power BI** for interactive business intelligence dashboards. The goal is to transform raw sales data into meaningful insights that support business decision-making.

The project demonstrates the complete analytics workflow, from writing SQL queries to designing an interactive dashboard.

---

## Objectives

- Analyze overall sales and profit performance.
- Identify high-performing regions, states, and cities.
- Evaluate sales across product categories and shipping modes.
- Monitor yearly sales trends.
- Build an interactive dashboard for business users.

---

## Tools & Technologies

- **PostgreSQL** – Data analysis using SQL
- **Power BI** – Dashboard development
- **Power Query** – Data transformation
- **DAX** – KPI calculations and measures

---

## SQL Analysis

The SQL analysis answers business questions including:

- Total Sales and Total Profit
- Highest Sales by Region
- Sales by State
- Sales by City
- Category Performance
- Yearly Sales Trends
- Shipping Mode Analysis
- Customer Insights
- Discount Analysis
